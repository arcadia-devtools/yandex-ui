# Spacer



<!-- description:start -->
Компонент который вставляет своих детей с заданным смещением.
<!-- description:end -->

## Использование компонента

Значения внутренних отступов можно задать с помощью 3 вариантов:

1. С помощью свойства `all`:

  ```ts
  // src/App.ts
  import React from 'react'
  import { Spacer } from '@yandex/ui/Spacer'

  const App = () => (
    <Spacer all={10}>
      Content
    </Spacer>
  )
  ```

2. С помощью свойства `vertical` и/или `horizontal`:

  ```ts
  // src/App.ts
  import React from 'react'
  import { Spacer } from '@yandex/ui/Spacer'

  const App = () => (
    <Spacer vertical={10} horizontal={20}>
      Content
    </Spacer>
  )
  ```

3. С помощью свойства `top` и/или `bottom`, и/или `right`, и/или `left`:

  ```ts
  // src/App.ts
  import React from 'react'
  import { Spacer } from '@yandex/ui/Spacer'

  const App = () => (
    <Spacer top={10}>
      Content
    </Spacer>
  )
  ```

> Указать единицы измерения можно передав значение в виде строки, например — `10rem`, по умолчанию будет использованы `px`.
